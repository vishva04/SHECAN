

# She Can Foundation Website

## Overview

The **She Can Foundation Website** is a responsive and visually engaging landing page designed to promote women's empowerment and showcase the mission of the She Can Foundation.

The website features a modern UI with smooth animations, an interactive image carousel, informative content sections, and a clean user experience.

## Features

* Responsive design for desktop and mobile devices
* Interactive image carousel
* Modern and accessible user interface
* Smooth animations and transitions
* About section highlighting the NGO's mission
* Call-to-action button
* Professional footer section
* Lightweight and fast-loading static website

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
